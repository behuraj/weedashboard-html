# WEE Index Dashboard - New Design

A modern, clean dashboard design for WEE Index Dashboard.

## Project Structure

```
new-dashboard/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # Compiled CSS
├── scss/
│   └── style.scss     # SCSS source file
├── js/
│   └── main.js        # JavaScript functionality
└── README.md          # This file
```

## Features

- **Top Header**: Blue gradient header with logo, dashboard title, and profile dropdown
- **Sidebar**: Hover menu layout with navigation items
- **8 Stat Cards**: Modern card design with icons and metrics
- **Map Section**: Left side image display
- **District Ranking**: Right side district-wise ranking with 6 indicators

## Usage

1. Open `index.html` in a web browser
2. The design is responsive and works on different screen sizes

## Browser Support

- Chrome (latest)
- Firefox (latest)
- Safari (latest)
- Edge (latest)

## Notes

- Update image paths in `index.html` to match your actual image locations
- Customize colors in CSS variables at the top of `style.css`
- Add more districts in the ranking section as needed
